# Photos des flacons

Dépose ici les photos de chaque parfum. Vite sert ce dossier tel quel
depuis la racine du site (`/images/flacons/...`).

## Noms de fichiers attendus

| Collection     | Fichier                  | Utilisé dans                          |
|-----------------|--------------------------|----------------------------------------|
| Arctique        | `arctique.jpg`           | index.html, collections.html, gallery.html |
| Cristal Noir    | `cristal-noir.jpg`       | index.html, collections.html, gallery.html |
| Brume Blanche   | `brume-blanche.jpg`      | index.html, collections.html, gallery.html |
| Nocturne        | `nocturne.jpg`           | collections.html, gallery.html         |
| Solstice        | `solstice.jpg`           | collections.html, gallery.html         |
| Éclat           | `eclat.jpg`              | collections.html, gallery.html         |

## Recommandations

- Format portrait, ratio ~3:4 (ex. 1200 × 1600 px)
- JPG ou WebP, qualité 80-85 %
- Poids cible : < 250 Ko par image

Une fois les fichiers déposés ici, dis-le moi et je mets à jour les
balises `<img>` dans index.html, collections.html et gallery.html pour
pointer vers `/images/flacons/<nom>.jpg` à la place des URLs Unsplash.
